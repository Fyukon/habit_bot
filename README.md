# Habit Tracker Bot

Telegram-бот для отслеживания привычек.  
Реализовано:
- Добавление и удаление привычек
- FSM-механизм ввода
- Inline и Reply клавиатуры
- База данных SQLite
- Интеграция с Dog API

## Установка

```bash
git clone https://github.com/USERNAME/habit-bot.git
cd habit-bot
pip install -r requirements.txt
